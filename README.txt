Instructions pour mettre en ligne le site DiasporAveniR sur GitHub Pages :

1. Crée un compte sur https://github.com si ce n’est pas déjà fait.
2. Crée un nouveau dépôt (repository), par exemple : diasporavenir-site
3. Clique sur "Add file" > "Upload files" et envoie les fichiers index.html et style.css
4. Une fois les fichiers envoyés, clique sur "Commit changes"
5. Va dans "Settings" > "Pages"
6. Sous "Source", choisis la branche `main` et le dossier `/root`, puis clique sur "Save"
7. Ton site sera accessible à l’adresse : https://tonpseudo.github.io/diasporavenir-site/

Personnalise le contenu de index.html selon les besoins de ton association.
